## üõ∞Ô∏è Data Sources

### NASA APIs Integration
Our dashboard integrates with three primary NASA data sources:

#### 1. APOD (Astronomy Picture of the Day) API
- **Source**: NASA Goddard Space Flight Center
- **Update Frequency**: Daily
- **Content Type**: Educational imagery and explanations
- **Historical Data**: Available since 1995

#### 2. DONKI API (Space Weather Database)
- **Source**: NASA's Space Weather Database of Notifications, Knowledge, Information
- **Update Frequency**: Real-time to hourly
- **Content Type**: Solar flares, coronal mass ejections, geomagnetic storms
- **Coverage**: Comprehensive space weather events

#### 3. NeoWs API (Near Earth Object Web Service)
- **Source**: NASA's Center for Near-Earth Object Studies (CNEOS)
- **Update Frequency**: Daily updates
- **Content Type**: Asteroid close approaches, orbital data
- **Coverage**: All known near-Earth objects
